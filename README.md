# STOC 2026 Workshop Website

Static website for **Building New Algorithmic Foundations for Cryptography**.

## Preview locally

From this folder, run:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Files

- `index.html` — page content and metadata
- `styles.css` — responsive layout, visual design, and animations
- `script.js` — mobile navigation and reveal behavior
- `assets/algorithmic-horizon.png` — generated workshop hero artwork

No build step or external JavaScript dependencies are required.
